Currículo Online - Eduarda Jammily
Sobre o Projeto

Este projeto foi desenvolvido como atividade prática da disciplina de Desenvolvimento Web.

O objetivo é criar um currículo online utilizando apenas HTML, CSS e JavaScript, sem o uso de frameworks ou bibliotecas externas.

O site apresenta informações pessoais, formação acadêmica, portfólio e um formulário de contato com validação em JavaScript.

Funcionalidades
Menu de navegação entre seções.
Seção "Sobre Mim".
Seção "Formação".
Seção "Portfólio".
Formulário de contato com validação de campos.
Alternância entre tema claro e escuro utilizando JavaScript.
Layout responsivo para diferentes tamanhos de tela.
Tecnologias Utilizadas
HTML5
CSS3
JavaScript
